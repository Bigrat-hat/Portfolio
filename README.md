# Personal Portfolio — Aditya

Responsive portfolio website built with React.js.

## Tech Stack
React.js · JavaScript · HTML5 · CSS3 · Vite · Git · GitHub

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build for Production

```bash
npm run build
```
